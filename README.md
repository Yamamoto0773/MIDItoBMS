# MIDItoBMS
MIDIデータから自作音ゲーの譜面データに変換するツール


### コミットメッセージの書式
1行目:コミットの種類

2行目:改行

3行目:コミットの要約


### コミットの種類
add     新規追加

fix     バグ修正

update  バグではない変更

disable コメントアウトなど、機能の無効化

remove  ファイルやコードの削除

clean   ファイルやコードの整理
